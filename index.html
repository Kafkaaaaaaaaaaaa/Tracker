Database Setup Instructions

To use this tracker without Firebase, we are using JSONBin.io, a free, lightweight cloud storage service.

  1. Go to JSONBin.io and create a free account.
  2. Once logged in, click "Create New" to make a new empty bin. Add `{ "logs": [] }` as the initial data and save it.
  3. Copy the Bin ID from the URL (e.g., 6789abcd...).
  4. Go to your API Keys page and copy your Master Key.
  5. Open this HTML file in an editor and replace YOUR_JSONBIN_MASTER_KEY and YOUR_BIN_ID around line 125.
